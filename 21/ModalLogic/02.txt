# Полнота модальных логик

Теория - набор формул.

Если L - логика, то теория \Gamma L-противоречива,
если \exists \phi_1 .. \phi_n, ! (\phi_1 /\ .. /\ \phi_n) \in \L.

Теория \Gamma - максимально непротиворечива,
если \forall \Sigma (непротиворечивый) -> \Gamma = \Sigma.

Теория \Gamma полная, если \forall \phi, \phi \in \Gamma \/ ! \phi \in \Gamma.

Теория полна <=> когда она максимально непротиворечива.

Лемма (Линденбаума).
  Пусть \Gamma - непротиворечивая теория. Тогда \Gamma содержится в полной теории.

Каноническая шкала логики L:
  F_L = (W_L, R_L), где
  W_L - множество L-полных теорий
  \Gamma R_L \Delta <=> \forall \phi, \phi \in \Delta => <>\phi \in \Gamma.

Каноническая модель: канон. шкала + канон. оценка:
  \Theta_L(p) = { \Gamma \in W_L : p \in \Gamma }

Лемма.
  M_L |= \phi <=> \phi \in L.

Теорема.
  K = Log(FF), где FF - класс всех шкал.
(доказательство - через каноническую шкалу).

Аналогичные теоремы есть и для других логик и соответствующих им классов шкал.

Логика L - каноническая, если L = Log(F_L).
